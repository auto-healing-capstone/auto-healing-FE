
# User Dashboard

React + Vite based user dashboard UI.

The original visual source came from the Figma file at `https://www.figma.com/design/7YTrVb4SSEYxhdOtgiFq6C/User-dashboard`.

## Recommended structure

```text
src/
  app/         # app bootstrap
  features/    # screen-level feature folders
  layouts/     # shared page shells
  routes/      # router config
  shared/      # reusable UI and shared helpers
  styles/      # global styles and theme files
```

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
